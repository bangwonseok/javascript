1. 산술연산자
	사칙연산(+, -, *, /)
	나머지연산 %
	홀수  - 2로 나눈 나머지가 1인 경우
		a % 2 == 1 --> 홀수
	짝수 - 2로 나눈 나머지가 0인 경우
		a % 2 == 0 --> 짝수

		text += "반갑습니다.";
		
3. 대입연산자 (=)
	var a=1;

4. 복합 대입연산자
	(산술연산 + 대입연산자)
	a= a+2; // a+=2;
	a= a-2 // a-=2;
	a= a/2 // a/=2;
